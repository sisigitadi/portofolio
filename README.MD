About This Project

This portfolio is a single, self-contained index.html file. It's designed to be lightweight, fast, and fully responsive, showcasing my dual expertise in Artificial Intelligence and Cybersecurity.

It was built from scratch and iteratively improved to be modern, futuristic, and highly interactive, providing a personal and engaging experience for visitors.

Tech Stack

This project is intentionally lean and demonstrates mastery of foundational web technologies without relying on a heavy framework like React or Vue.

HTML5: For semantic structure.

Tailwind CSS: For all utility-first styling and responsiveness.

Vanilla JavaScript (ES6+): For all interactivity, including:

Dynamic typing effects

Scroll-based animations (fade-ins, counters)

Project filtering logic

3D card flip effects

Modal/pop-up system

Floating contact widget

D3.js: To create the interactive force-directed bubble map for the "Skills" section.

Formspree: For serverless handling of the contact form.

RSS2JSON API: To dynamically fetch and display the latest articles from my Medium blog.

Key Features

Interactive Particle Background: A mouse-interactive particle field in the hero section.

Dynamic 3D Flip Cards: Project cards that flip on hover to reveal case study details.

Interactive Skill Map: A D3.js force-directed graph visualizing my dual specialties.

Live Medium Feed: Automatically fetches my latest blog posts.

"Scrambling" Text Effect: Section headings animate into view with a futuristic scramble effect.

Animated Stat Counter: Key stats (like "10+ Years") animate on scroll.

Floating Contact Widget: A persistent, clean, and modern way to get in touch.

Tonal Dark Mode: A single, professional dark-mode theme designed for clarity and depth.

How to Deploy (GitHub Pages)

Deploying this site is simple:

Create a Repository: Create a new public repository on GitHub (e.g., portfolio or your-username.github.io).

Upload Files: Upload the index.html and README.md files from this project to your new repository.

Enable Pages:

In your repository, go to Settings > Pages.

Under "Build and deployment," select the Branch you uploaded to (e.g., main).

Select the /(root) folder and click Save.

Wait & View: Wait about 1-2 minutes. Your site will be live at https://[YOUR_USERNAME].github.io/[YOUR_REPO_NAME]/.

(If you named your repository your-username.github.io, the site will be live at https://your-username.github.io/)
